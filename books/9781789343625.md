## Chapter 15/23

Create a joinable thread.

* [pthread\_create]()
* [pthread\_join]()

<details>
<summary>code sample</summary>

```c
#include <stdio.h>
#include <errno.h>
#include <pthread.h>

void* handler(void*);

int main(void)
{
    pthread_t thread;

    if (pthread_create(&thread, NULL, handler, NULL))
        return(errno);

    if (pthread_join(thread, NULL))
        return(errno);
}

void* handler(void* arg)
{
    (void)arg;
    printf("working thread\n");
    return NULL;
}
```
